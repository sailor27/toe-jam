## Project Description
Toe Jam is an interactive Tic-Tac-Toe game built using React.


## Built Using:

* Webpack
* Javacript
* React.js

## Installation Instructions
in the terminal:
```

npm install webpack@3.4.0 webpack-dev-server@2.5.0 eslint eslint-plugin-react -g

clone this repository
npm init
npm install
```
